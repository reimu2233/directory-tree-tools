### 目录树_基础版.py

**用途**
生成基础目录结构的树形图文本。

**核心功能**

* **零依赖**：纯 Python 标准库实现，无需任何第三方库。
* **遍历机制**：调用 `os.scandir` 执行目录扫描，按目录优先、文件名小写排序输出。
* **路径兼容**：支持带特殊字符及 Unicode 编码的路径，遇权限不足或 I/O 错误时自动跳过并提示。
* **状态反馈**：控制台采用 ANSI 颜色输出，成功显示绿色耗时与条目数，失败显示红色错误信息。
* **结果输出**：在根目录生成 `目录树.txt`（UTF-8 BOM 编码），首行展示 `根目录名（完整路径）`。

**使用方法**

* 双击运行：扫描脚本所在目录。
* 命令行运行：`python 目录树_基础版.py "目标路径"`。

### 目录树_高权限版.py

**用途**
请求系统管理员权限以遍历受限目录，解决“拒绝访问”问题。

**核心功能**

* **自动提权**：内置 UAC 提权逻辑，若未以管理员身份运行则自动申请提权。
* **智能存储**：若扫描目标位于 C 盘，结果将自动导出至 `E:\` 根目录，避免因权限限制无法在系统盘根目录写入文件。
* **鲁棒性**：针对 C 盘敏感目录优化，遇权限不足自动记录警告并继续扫描。
* **状态反馈**：控制台区分展示 [管理员] 或 [普通权限] 扫描状态。

**使用方法**

* 双击运行：触发 UAC 授权后扫描脚本所在目录。
* 命令行运行：`python 目录树_高权限版.py "目标路径"`。

### 目录树_高级版.py

**用途**
生成指定目录的树形结构文本，并统计音视频文件时长。

**核心功能**

* **时长检测**：优先调用 `ffprobe`（采用 `csv` 输出格式直取数值，省去 JSON 解析开销），其次回退 `tinytag` 库读取音视频时长。
* **并发处理**：使用 32 线程的 `ThreadPoolExecutor` 并发探测媒体时长，控制台实时刷新进度。
* **增量缓存**：按“路径+大小+修改时间”缓存时长结果至本地缓存目录，工作目录无残留，重复扫描秒出结果。
* **状态反馈**：控制台 ANSI 颜色输出，显示耗时、文件数、引擎状态及缓存命中率。
* **详尽报告**：生成 `目录树_详细.txt`，包含音视频总时长统计、文件类型分布及带 `[hh:mm:ss]` 标注的树形结构。

**使用方法**

* 双击运行：扫描脚本所在目录。
* 命令行运行：`python 目录树_高级版.py "目标路径"`。
* *注：建议在系统环境变量中配置 ffmpeg/ffprobe，或执行 `pip install tinytag` 以启用时长检测。*
