# Directory Tree Tools 🌳

Windows 环境下的多功能目录树生成工具集，支持常规扫描、UAC 提权越权扫描以及带音视频时长探测的高级并发扫描。

## 🛠️ 工具集简介

本仓库包含三个独立的 Python 脚本，用于应对不同场景下的目录结构导出需求。所有的脚本均针对 Windows 环境进行了优化处理（如 ANSI 颜色输出、控制台编码等）。

### 1. 🌱 目录树_基础版.py (Basic Version)
**主打一个轻量、无依赖、开箱即用。**
* **零依赖**：纯 Python 标准库实现，无需安装任何第三方包。
* **鲁棒性**：底层调用 `os.scandir`，完美支持带方括号等特殊字符的路径，遇权限不足或 I/O 错误时自动跳过并红字警告，不中断主流程。
* **输出清晰**：在根目录生成 `目录树.txt`（UTF-8 BOM 编码，防止 Windows 记事本乱码）。

### 2. 🛡️ 目录树_高权限版.py (Elevated Version)
**专治系统盘各种“拒绝访问”，智能路由防污染。**
* **UAC 自动提权**：运行即自动检测管理员权限，若无权限则触发 UAC 弹窗要求重启自身提权。
* **系统目录畅通无阻**：获取管理员权限后，可无视大部分 `PermissionError`，完整遍历受保护的系统级目录。
* **智能路径决策**：
    * 当扫描目标在 `C盘`（系统盘）时，为防止污染系统目录，结果会自动写入 `E:\目录树_<目录名>_<时间戳>.txt`。
    * 若 E 盘不可用，则自动回退到扫描目录原地写入。

### 3. 🚀 目录树_高级版.py (Advanced Version)
**强迫症福音，带媒体时长探测的满血版。**
* **时长侦测引擎**：优先调用系统环境变量中的 `ffprobe`（采用原生 CSV 格式极速读取），若未安装则回退使用 `tinytag` 库。
* **32 线程并发**：采用 `ThreadPoolExecutor` 对音视频文件（支持数十种主流格式）进行多线程时长探测，告别龟速。
* **哈希增量缓存**：扫描过的文件时长会基于 `路径+大小+修改时间` 缓存在 `%LOCALAPPDATA%\目录树缓存` 中。同一目录二次扫描“秒出”。
* **详尽数据报告**：输出 `目录树_详细.txt`，头部附带详尽的统计报表（文件总数、视频总时长、音频总时长、各后缀名数量统计），树状图中音视频文件后方会追加 `[hh:mm:ss]` 时长标注。

## 🚀 快速开始

### 运行环境
* **操作系统**: Windows 10/11
* **Python**: Python 3.7+ (推荐 3.10+)

### 依赖安装 (仅高级版需要)
基础版和高权限版**不需要**任何配置。针对 `高级版`，建议配置以下环境之一以开启时长探测：
1.  **推荐配置**：安装并配置 [FFmpeg](https://ffmpeg.org/)（确保 `ffprobe` 在系统 PATH 环境变量中）。
2.  **备选配置**：或者安装纯 Python 库 `tinytag`：
    ```bash
    pip install tinytag
    ```

### 使用方法
本工具集支持极其“傻瓜式”的操作逻辑：

* **方式一：双击运行 (GUI 懒人法)**
    将脚本文件直接放在你需要扫描的文件夹内，双击运行，按提示操作即可完成当前目录的扫描。
* **方式二：命令行 (Geek 玩法)**
    打开 CMD 或 PowerShell，通过参数传入目标路径：
    ```bash
    python 目录树_基础版.py "D:\我的学习资料"
    python 目录树_高权限版.py "C:\Program Files"
    ```
